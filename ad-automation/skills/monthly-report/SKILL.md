# 月次レポートスキル

## 目的
クライアント提出用の月次レポートを、全社様式（`CLAUDE.md §6`）とトーン（§7）で生成する。

## 手順
1. `CLAUDE.md` → `clients/<社名>.md` を読む（目標値・トーン&マナー・禁止表現）。
2. 当月の 費用 / CV / CPA / ROAS を取得（読み取り）。前月・目標と対比（増減率付き）。
3. 構成（§6）：サマリー → KPIハイライト → 要対応 → 提案(下書き) → 注記。
4. 文体は敬体・事実ベース・断定回避（§7）。医療広告の禁止表現チェック（§3.2）。
5. 数値には必ず期間・母数を添える。推測は「推定」と明記。

## 出力
- `out/report_<社名>_<年月>.md`（提出前に人間レビュー必須）。

## 禁止事項
- 目標達成の断定・効果保証表現。数値の捏造・欠損の穴埋め（「データなし」と書く）。
